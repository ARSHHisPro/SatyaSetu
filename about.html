🏛️ Project SatyaSetu

SDG Goal 16 Educational Portal Initiative

Goal & Purpose

SatyaSetu is designed as an educational, offline-compatible simulation portal modeling civic integrity reporting pathways. By documenting infrastructure, resource allocation, and civic transparency metrics, we raise awareness of Sustainable Development Goal 16 (Peace, Justice, and Strong Institutions).

Core Features & Integrations

  • Offline Compatibility: Fully operates without internet backups using Service Worker caches and LocalStorage fallbacks.
  • On-Device NLP Analysis: Automatically flags keyboard spammers, profanities, and duplicate entries entirely client-side.
  • Interactive Chart Dashboards: Uses Chart.js to map severity thresholds, classifications, and daily trends.
  • PII Audit Shield: Local regular expressions shield user reports from leakage of sensitive credential tokens.

Technology Stack

Built with vanilla standard HTML5, CSS3 styling variables, vanilla JavaScript global scopes, Firebase compatibility libraries, Chart.js, Lucide Icons, and ElevenLabs conversational voice assistants.
